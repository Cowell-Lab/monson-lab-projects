# MAV

---

## Files

- make_jobs.ipynb : Jupyter notebook to create job files. Requires `airr` and `./repertoires.airr.json`.
- make_jobs.ip : Contains function used in the above notebook.
- NOTES.md : Contains notes about jobs.
- run-jobs.sh : Used for submitting jobs. Run with the command `sh ./run-jobs.sh igblast` to submit all jobs in the current directory (excluding a subdirectory named `./old_jobs`).
